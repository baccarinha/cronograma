# Projeto de Escala de Frente de Caixa

O projeto Gerenciamento de Escala da Frente de Caixa é um sistema web desenvolvido para organizar e controlar escalas de trabalho de funcionários, especialmente voltado para ambientes como supermercados ou lojas com múltiplos setores. O sistema funciona como um painel administrativo (dashboard) que permite cadastrar colaboradores, definir horários, organizar folgas e gerar cronogramas de forma estruturada e visual.

A proposta do projeto é facilitar o gerenciamento de equipes, reduzindo erros manuais e permitindo uma visualização rápida das escalas de trabalho. O sistema apresenta um painel inicial que exibe informações importantes, como total de funcionários cadastrados, número de colaboradores trabalhando no dia, setores ativos e cronogramas criados, oferecendo uma visão geral da operação.

Um dos principais recursos do sistema é o cadastro de funcionários, onde é possível inserir dados como nome completo, setor de atuação, horário de trabalho e dia de folga semanal. O sistema também permite definir ciclos de folga aos domingos, organizando automaticamente períodos alternados de descanso, o que é comum em escalas rotativas.

Além disso, o projeto possui a funcionalidade de criação de cronogramas, permitindo organizar turnos de trabalho com base nos funcionários cadastrados. O usuário pode visualizar os cronogramas gerados e exportá-los em formato PDF, facilitando o compartilhamento das escalas com a equipe. Também há recursos para importar e exportar dados, permitindo salvar ou reutilizar informações quando necessário.

Outro recurso importante é o gerenciamento de setores, onde é possível criar diferentes áreas de trabalho, definir cores para identificação visual e adicionar ícones personalizados. Essa funcionalidade melhora a organização e torna o sistema mais intuitivo e visualmente amigável.

Do ponto de vista técnico, o projeto demonstra a aplicação de tecnologias web modernas, com estrutura baseada em HTML, CSS e JavaScript, além de recursos interativos típicos de sistemas administrativos, como formulários dinâmicos, painéis visuais e geração de relatórios em PDF. O sistema está hospedado online por meio do GitHub Pages, permitindo acesso fácil pela internet sem necessidade de instalação.

🧩 Resumo técnico do projeto

Nome do projeto:
Gerenciamento de Escala da Frente de Caixa

Objetivo:
Criar um sistema digital para organizar escalas de funcionários e controlar cronogramas de trabalho.

Principais funcionalidades:

Cadastro de funcionários
Definição de horários de trabalho
Controle de dias de folga
Organização por setores
Criação e visualização de cronogramas
Exportação e importação de dados
Geração de cronogramas em PDF
Dashboard com indicadores gerais
